Functional Programming¶

calcpy provides APIs for function execution and argument reordering.

Execute Function¶

calcpy.call(f, *args, **kwargs)[source]¶

Call a callable with positional arguments and keyword arguments.

Parameters:

  • f – Callable object.

  • *args – Positional arguments.

  • **kwargs – Keyword arguments.

Returns:

Result of the callable.

Examples

>>> call(range, 2, 3, 6)
range(2, 3, 6)

calcpy.cyclepermcaller(f, /, cycle=())[source]¶

Callable that swaps position parameters according to cyclc notation.

Parameters:

  • f (callable)

  • cycle (list | tuple) – List of indices to swap.

Return type:

callable

Examples

>>> executor = cyclepermcaller(range, cycle=[0, 1])
>>> executor(3, 2, 6)
range(2, 3, 6)

>>> executor = cyclepermcaller(range, cycle=[1, 2])
>>> executor(3, 2, 6)
range(3, 6, 2)
calcpy.swapcaller(f, /, i=0, j=1)[source]¶

Callable that swaps positional arguments in a pair.

Parameters:

  • f (callable)

  • i (int) – Index of the argument to swap.

  • j (int) – Index of another argument to swap.

Return type:

callable

Examples

>>> executor = swapcaller(range)
>>> executor(3, 2, 6)
range(2, 3, 6)

>>> executor = swapcaller(range, i=1, j=2)
>>> executor(3, 2, 6)
range(3, 6, 2)
